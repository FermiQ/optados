# `OptaPyDOS.py` - Python Interface for OptaDOS Fortran Modules

## Overview

`OptaPyDOS.py` is a Python module that serves as a direct interface to the compiled Fortran modules of the OptaDOS library. This file is automatically generated by `f90wrap`, a tool that creates Python wrappers for Fortran code. It allows Python users to call Fortran subroutines and functions as if they were native Python methods and access Fortran module variables.

The structure of `OptaPyDOS.py` typically consists of Python classes that mirror the Fortran modules (e.g., `Od_Algorithms` class for `od_algorithms` Fortran module). These classes contain static methods and properties that map to the Fortran procedures and module variables, respectively.

## Key Components

This file defines several classes, each corresponding to a Fortran module in OptaDOS. Below are some of the primary classes and a general idea of their structure:

*   **`class Od_Algorithms(f90wrap.runtime.FortranModule)`**:
    *   Wraps the `od_algorithms` Fortran module.
    *   Provides methods like `channel_to_am()`, `gaussian()`, `heap_sort()`, `utility_lowercase()`, `utility_cart_to_frac()`, `utility_frac_to_cart()`, `algorithms_erf()`.
    *   The docstrings for these methods are usually derived from the comments in the corresponding Fortran source file (`algorithms.f90`).

*   **`class Od_Cell(f90wrap.runtime.FortranModule)`**:
    *   Wraps the `od_cell` Fortran module.
    *   Provides methods like `cell_find_mp_grid()`, `cell_get_symmetry()`, `cell_read_cell()`, `cell_calc_lattice()`, etc.
    *   Exposes module variables like `real_lattice`, `recip_lattice`, `cell_volume`, `nkpoints`, `num_atoms` as properties.

*   **`class Od_Comms(f90wrap.runtime.FortranModule)`**:
    *   Wraps the `od_comms` Fortran module (likely handling MPI communications).
    *   Provides methods for communication operations like `comms_setup()`, `comms_end()`, `comms_bcast()`, `comms_send()`, `comms_recv()`, `comms_reduce()`.
    *   Exposes variables like `on_root`, `num_nodes`, `my_node_id`.

*   **`class Od_Constants(f90wrap.runtime.FortranModule)`**:
    *   Wraps the `od_constants` Fortran module.
    *   Exposes constants like `optados_version`, `copyright`, `dp` (double precision kind), `pi`, `h2ev`, `bohr2ang`.

*   **(Other classes for `Od_Core`, `Od_Dos`, `Od_Dos_Utils`, `Od_Electronic`, `Od_Io`, `Od_Jdos`, `Od_Jdos_Utils`, `Od_Optics`, `Od_Parameters`, `Od_Pdos`, `Xmgrace_Utils`)**:
    *   Each class wraps its corresponding Fortran module (e.g., `od_core`, `od_dos`, etc.).
    *   They expose the public subroutines, functions, and module variables from the Fortran side as Python methods and properties.
    *   The specific methods and variables available depend on the public interface of each Fortran module.

Additionally, the file may define top-level functions that wrap Fortran procedures not part of a specific module, or helper functions generated by `f90wrap`. Examples seen: `my_dcopy`, `my_zcopy`, `my_icopy`.

## Important Variables/Constants

*   The primary "variables" in this file are the class instances created for each wrapped module, e.g., `od_algorithms = Od_Algorithms()`. These instances are the entry points to access the Fortran module's contents.
*   Within each class, module variables from Fortran are exposed as Python properties (e.g., `od_cell.nkpoints`).

## Usage Examples

To use the Fortran routines from Python, one would typically import the specific module instance:

```python

import numpy as np

# Using a function from od_algorithms
mean = 0.0
width = 1.0
x_val = 0.5
gauss_val = od_algorithms.gaussian(m=mean, w=width, x=x_val)
print(f"Gaussian value: {gauss_val}")
```

## Dependencies and Interactions

*   **`_OptaPyDOS`**: This is the low-level extension module (likely a `.so` or `.pyd` file) compiled from the Fortran code and `f90wrap` generated C wrappers. `OptaPyDOS.py` calls into this binary module.
*   **`f90wrap.runtime`**: Provides the `FortranModule` base class and other runtime utilities needed for the wrapper to function.
*   **`logging`**: Standard Python logging module, potentially used by `f90wrap` or the wrappers for messages.
*   **Fortran Source Files**: The functionality of `OptaPyDOS.py` is entirely dependent on the content and structure of the original Fortran source files (`.f90`, `.F90`) located in the `../src/` directory (relative to where `f90wrap` was run, typically meaning the `optados/src` directory).

This Python file acts as a bridge, making the compiled Fortran library accessible and usable within a Python environment.
